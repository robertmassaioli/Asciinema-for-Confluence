# Marketplace Release Notes — Initial Release (v1.0.0)

Three options for the release notes field on the Atlassian Marketplace listing.
Limit: 1,000 characters. Character counts shown for each option.

---

## Option A — Feature-list style, CastScript first
*(698 chars)*

```
Initial release of Asciinema for Confluence.

CastScript Macro — Write terminal demos as plain text: describe commands, expected
output, chapter markers, and timing. The macro compiles your script to a live,
interactive terminal animation in the browser. Update a demo by editing text —
no re-recording required.

Recording Macro — Reference a .cast file attached to the current Confluence page.
The player loads the recording directly from the attachment.

Inline Macro — Paste raw .cast content into the macro body for an instant embedded
player with no attachment management.

Smart Links — Paste any asciinema.org URL to get an automatic rich preview card
showing the recording title, author, and duration.

All macros support theme, speed, autoplay, loop, and poster frame configuration.
No external data storage. Minimal permissions.
```

---

## Option B — Narrative style, problem-first
*(722 chars)*

```
Initial release of Asciinema for Confluence.

Terminal documentation has a maintenance problem: recordings go stale the moment
a CLI changes. This app fixes that.

CastScript Macro — Describe your terminal session as plain text and publish it as
a live animation. Edit the script when things change — no new recording needed.

Recording Macro — Embed a .cast file stored as a Confluence page attachment. The
recording lives alongside the documentation that describes it.

Inline Macro — Paste raw .cast content directly into the macro body. Zero file
management, instant player.

Smart Links — Drop any asciinema.org URL into a page for a rich preview card with
title, author, and duration.

Player configuration: five themes (asciinema, monokai, solarized-dark,
solarized-light, tango), speed control, autoplay, loop, and poster frames.
```
