# Macros

Asciinema for Confluence provides three macros and a Smart Link resolver.
Each one is a different way to get a terminal recording into your Confluence page.

| Macro | Best for |
|:------|:---------|
| [CastScript]({% link macros/castscript.md %}) | Writing demos as plain text — no recording required |
| [Recording]({% link macros/recording.md %}) | Embedding a `.cast` file attached to the page |
| [Inline]({% link macros/inline.md %}) | Pasting raw `.cast` content directly |
| [Smart Links]({% link macros/smart-links.md %}) | Rich preview cards for `asciinema.org` URLs |
